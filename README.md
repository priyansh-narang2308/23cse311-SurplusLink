# SurplusLink

A smart food redistribution platform that connects surplus food donors (like restaurants and event venues) with NGOs and shelters. The system uses ML-based matching, real-time tracking, and automated logistics to reduce food waste and hunger.

## Key Features

- Role-based login (Donor, NGO, Admin)
- Post and track food donations with expiry details
- ML-powered NGO matching based on location and need
- Real-time pickup and delivery tracking
- Hygiene checklist and delivery confirmation
- Dashboards with reports (meals saved, COâ‚‚ reduced)

## Tech Stack

- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Firebase (real-time)
- **DevOps & Tools:** GitHub Actions, Docker, AWS
- **Testing:** Postman, Jest, and Vitest
